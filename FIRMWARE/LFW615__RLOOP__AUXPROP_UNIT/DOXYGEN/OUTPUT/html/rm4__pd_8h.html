void pmmInit ( void  )

The PMM provides memory-mapped registers that control the states of the supported power domains.

The PMM includes interfaces to the Power Mode Controller (PMC) and the Power State Controller (PSCON). The PMC and PSCON control the power up/down sequence of each power domain.
