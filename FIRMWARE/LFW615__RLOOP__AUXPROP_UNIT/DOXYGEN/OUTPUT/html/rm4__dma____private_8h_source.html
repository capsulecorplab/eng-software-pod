21  /* 16.2.3 Addressing Modes
22  There are three addressing modes supported by the DMA controller that can be setup independent for the
23  source and the destination address:
24  * Constant -- source and/or destination addresses do not change.
25  * Post incremented -- source and/or destination address are post-incremented by the element size.
26  * Indexed -- source and/or destination address is post-incremented as defined in the Element Index
27  Offset Register (Section 16.3.2.5) and the Frame Index Offset Register (Section 16.3.2.6).
28  An unaligned address with respect to the element size is not supported. An address chosen during
29  transfer must be divisible by the element size.
30  */
