void vRM4_EMAC_INT__CoreIntAck ( Luint32  u32EOIFlag)

Acknowledges an interrupt processed to the EMAC Control Core.

Acknowledges an interrupt processed to the EMAC Control Core. The application software must acknowledge the EMAC control module after processing packets by writing the appropriate C0TX key to the EMAC End-Of-Interrupt Vector register (MACEOIVECTOR). See Section 28.5.12 for the acknowledge key values.

Parameters
[in]u32EOIFlagu32EOIFlag can take the following values: EMAC_INT_CORE0_TX - Core 0 TX Interrupt EMAC_INT_CORE1_TX - Core 1 TX Interrupt EMAC_INT_CORE2_TX - Core 2 TX Interrupt EMAC_INT_CORE0_RX - Core 0 RX Interrupt EMAC_INT_CORE1_RX - Core 1 RX Interrupt EMAC_INT_CORE2_RX - Core 2 RX Interrupt
