void vRM4_EMAC_LINK__Init ( Luint8 *  pu8MAC,
Luint8 *  pu8IP 
)

Init the link state machine Call this at power up to get the EMAC working.

The EMAC controls the flow of packet data from the system to the PHY.

Example:
u8MACAddx[0] = 0x00; u8MACAddx[1] = 0x26; u8MACAddx[2] = 0x80; u8MACAddx[3] = 0x00; u8MACAddx[4] = 0x00; u8MACAddx[5] = 0x01;

u8IPAddx[0] = 192; u8IPAddx[1] = 168; u8IPAddx[2] = 1; u8IPAddx[3] = 200;
