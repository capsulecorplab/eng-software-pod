1 /**
2  * @file RM4_SYSTEM__PMU.H
3  * @brief PMU header
4  * Performance Measurement Unit (PMU) Cycle Count (Cortex-R4F only)
5  * The PMU cycle count method uses the Cortex-R4F performance measurement unit (PMU) to count
6  * CPUCLK cycles during the time the measured code is running. The code execution time can then be
7  * calculated by multiplying the counted CPU clock cycles with the clock period.
8  * The PMU consists of four counters: one only counting clock cycles and three for counting selectable
9  * events. The example code use event counter 0 to count the clock cycles; so the PMU configuration can be
10  * easily changed to count any other event type (see 'enum pmuEvent' in the sys_pmu.h source code). Since
11  * the CPU runs with system frequency (CPU clock MCLK, fMCLK = fHCLK), HCLK is used as the timebase for
12  * the measurements. In the given code example, event counter 0 of the PMU is set up to count CPU clock
13  * cycles (Event = PMU_CYCLE_COUNT).
