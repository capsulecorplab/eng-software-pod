112  /*
113  No-acknowledge (NACK) mode
114  This bit is used to send an acknowledge (ACK) or a no-acknowledge (NACK) to the transmitter.
115  This bit is only applicable when the I2C is in receiver mode. In master receiver mode, when the
116  internal data counter decrements to zero, the I2C sends a NACK.
117  The master receiver I2C finshes a transfer when it sends a NACK. The I2C ignores ICCNT when NACKMOD is 1. The NACKMOD
118  bit should be set before the rising edge of the last data bit if a NACK must be sent, and this bit is
119  cleared once a NACK has been sent.
120  0 The I2C sends an ACK to the transmitter during the acknowledge cycle.
121  1 The I2C sends a NACK to the transmitter during the acknowledge cycle.
122  */
