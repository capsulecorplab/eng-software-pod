Lint16 s16RM4_I2C_USER__Rx_U8 ( Luint8  u8DeviceAddx,
Luint8  u8RegisterAddx,
Luint8 *  pu8Byte 
)

Receives a byte from a device.

Note
When passing in the device address, this is the I2C address that the device communicates over It is the NON SHIFTED version. So if you have a device address from the datasheet of 0xEF for read = 0b1110111R then the address you need to supply is actually 0x77 = 0b01110111.
Parameters
[out]*pu8BytePointer to Returned byte
[in]u8RegisterAddxRegister address on device
[in]u8DeviceAddxDevice hardware address on bus
Returns
0 = success
-ve = error
