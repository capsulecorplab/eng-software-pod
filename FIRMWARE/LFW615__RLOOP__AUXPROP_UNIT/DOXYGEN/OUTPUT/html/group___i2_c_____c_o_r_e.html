Luint16 u16RM4_I2C__IsTxBufferEmpty ( void  )

Is Tx Ready.

C_RM4I2C__IMR_REG__BIT__TXRDYEN : This bit is set to 1 to indicate when data in the transmit data register, I2CDXR, has been copied into the transmit shift register. This bit can also be polled by the device to indicate when to write the next transmitted data into the I2CDXR. Writing a 1 to this bit will set it.

#return 0 = I2CDXR contains data to transmit
1 = I2CDXR is empty.
