Detailed Description

Main implemention of software CRC.

https://en.wikipedia.org/wiki/Cyclic_redundancy_check Software based CRC (Cyclic Redundancy Check) This module implements the following CRC functions

  1. CRC16 using the CCITT algo. (x^16,x^12,x^5,x^1)

CRC7 using the SD CARD algo. (x^7 + x^3 + 1)

  1. CRC8 using the CRC8-CCITT algo

Due to the relevant speeds of various CPUs, a lookup table approach is also added so as we have faster LUT computation rather than the slower mathematical model.

Example:
An example of the CRC16 is to take a buffer of 10 bytes from 0x00 to 0x09. Compute the CRC over the buffer of 10 bytes will yield a result (example 0x0A8F).
