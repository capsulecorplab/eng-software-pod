Detailed Description

Handles the undervoltage threshold functions.

Note
A programmable undervoltage detection function is embedded in the ATA6870N. After being digitalized, each of the 6 voltages is compared to a programmable threshold defined in the UdvThresh register. If one of the six channels is out of the range defined by the threshold, an interrupt is requested to the host MCU if the corresponding udv mask bit is not set in the IrqMask register. The default threshold is 1.5V. As soon as MCU has acknowledged, undervoltage information is no more available to MCU, because status register is cleared when MCU reads it out. As a consequence, the next undervoltage interrupt cannot occur until the Atmel ATA6870N leaves its current undervoltage state.
