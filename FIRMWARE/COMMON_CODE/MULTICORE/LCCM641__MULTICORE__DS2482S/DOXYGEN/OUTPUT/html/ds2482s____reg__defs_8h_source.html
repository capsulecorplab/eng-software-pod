26  * Description Performs a global reset of device state machine logic. Terminates any ongoing 1-Wire

30  * Error Response None

34  * Status Bits Affected RST set to 1; 1WB, PPD, SD, SBR, TSB, DIR set to 0.

58  /**
59  * Write Configuration
60  * Command Code D2h
61  * Command Parameter Configuration Byte
62  * Description Writes a new configuration byte. The new settings take effect immediately. Note: When writing to
63  * the Configuration Register, the new data is accepted only if the upper nibble (bits 7 to 4) is the

65  * Typical Use Defining the features for subsequent 1-Wire communication.
66  * Restriction 1-Wire activity must have ended before the DS2482-100 can process this command.
67  * Error Response Command code and parameter are not acknowledged if 1WB = 1 at the time the command code

70  * acknowledge bit.

78  /**
79  * 1-Wire Reset
80  * Command Code B4h
81  * Command Parameter None
82  * Description Generates a 1-Wire reset/presence-detect cycle (Figure 4) at the 1-Wire line. The state of the
83  * 1-Wire line is sampled at tSI and tMSP and the result is reported to the host processor through the
84  * Status Register, bits PPD and SD.
85  * Typical Use To initiate or end any 1-Wire communication sequence.
86  * Restriction 1-Wire activity must have ended before the DS2482-100 can process this command. Strong

88  * is enabled, the PPD bit may not be valid and may cause a violation of the devices absolute
89  * maximum rating.
90  * Error Response Command code is not acknowledged if 1WB = 1 at the time the command code is received and
91  * the command is ignored.
92  * Command Duration tRSTL + tRSTH + maximum 262.5ns, counted from the falling SCL edge of the command code
93  * acknowledge bit.
94  * 1-Wire Activity Begins maximum 262.5ns after the falling SCL edge of the command code acknowledge bit.
95  * Read Pointer Position Status Register (for busy polling).
96  * Status Bits Affected 1WB (set to 1 for tRSTL + tRSTH), PPD is updated at tRSTL + tMSP, SD is updated at tRSTL + tSI.

101  /**
102  * 1-Wire Single Bit
103  * Command Code 87h
104  * Command Parameter Bit Byte
105  * Description Generates a single 1-Wire time slot with a bit value as specified by the bit byte at the 1-Wire
106  * line (see Table 2). A value of 0b generates a write-zero time slot (Figure 5); a value of 1b
107  * generates a write-one time slot, which also functions as a read-data time slot (Figure 6). In either
108  * case, the logic level at the 1-Wire line is tested at tMSR and SBR is updated.
109  * Typical Use To perform single-bit writes or reads at the 1-Wire line when single bit communication is
110  * necessary (the exception).
111  * Restriction 1-Wire activity must have ended before the DS2482-100 can process this command.
112  * Error Response Command code and bit byte are not acknowledged if 1WB = 1 at the time the command code is
113  * received and the command is ignored.
114  * Command Duration tSLOT + maximum 262.5ns, counted from the falling SCL edge of the first bit (MSB) of the bit byte.
115  * 1-Wire Activity Begins maximum 262.5ns after the falling SCL edge of the MSB of the bit byte.
116  * Read Pointer Position Status Register (for busy polling and data reading).
117  * Status Bits Affected 1WB (set to 1 for tSLOT), SBR is updated at tMSR, DIR (may change its state).
118  * Config Bits Affected 1WS, APU, SPU apply.

122  /**
123  * 1-Wire Write Byte
124  * Command Code A5h
125  * Command Parameter Data Byte
126  * Description Writes a single data byte to the 1-Wire line.
127  * Typical Use To write commands or data to the 1-Wire line. Equivalent to executing eight 1-Wire Single Bit
128  * commands, but faster due to less I2C traffic.
129  * Restriction 1-Wire activity must have ended before the DS2482-100 can process this command.

132  * Command Duration 8 x tSLOT + maximum 262.5ns, counted from falling edge of the last bit (LSB) of the data byte.
133  * 1-Wire Activity Begins maximum 262.5ns after falling SCL edge of the LSB of the data byte (i.e., before the data
134  * byte acknowledge). Note: The bit order on the I2C bus and the 1-Wire line is different (1-Wire: LSB
135  * first; I2C: MSB first). Therefore, 1-Wire activity cannot begin before the DS2482-100 has received
136  * the full data byte.
137  * Read Pointer Position Status Register (for busy polling).
138  * Status Bits Affected 1WB (set to 1 for 8 x tSLOT).
139  * Config Bits Affected 1WS, SPU, APU apply.

165  /**
166  * 1-Wire Triplet
167  * Command Code 78h
168  * Command Parameter Direction Byte
169  * Description Generates three time slots: two read time slots and one write time slot at the 1-Wire line. The type
170  * of write time slot depends on the result of the read time slots and the direction byte. The direction
171  * byte determines the type of write time slot if both read time slots are 0 (a typical case). In this
172  * case, the DS2482-100 generates a write-one time slot if = 1 and a write-zero time slot if = 0.
173  * See Table 3.
174  * * If the read time slots are 0 and 1, they are followed by a write-zero time slot.
175  * * If the read time slots are 1 and 0, they are followed by a write-one time slot.
176  * * If the read time slots are both 1 (error case), the subsequent write time slot is a write-one.
177  * Typical Use To perform a 1-Wire Search ROM sequence; a full sequence requires this command to be
178  * executed 64 times to identify and address one device.
179  * Restriction 1-Wire activity must have ended before the DS2482-100 can process this command.
180  * Error Response Command code and direction byte is not acknowledged if 1WB = 1 at the time the command
181  * code is received and the command is ignored.
182  * Command Duration 3 x tSLOT + maximum 262.5ns, counted from the falling SCL edge of the first bit (MSB) of the
183  * direction byte.
184  * 1-Wire Activity Begins maximum 262.5ns after the falling SCL edge of the MSB of the direction byte.
185  * Read Pointer Position Status Register (for busy polling).
186  * Status Bits Affected 1WB (set to 1 for 3 x tSLOT), SBR is updated at the first tMSR, TSB and DIR are updated at the
187  * second tMSR (i.e., at tSLOT + tMSR).
188  * Config Bits Affected 1WS, APU apply.
189  */
